# Conector de upload de arquivo local

O Adobe Experience Platform permite que os dados sejam assimilados de fontes externas e sistemas locais enquanto fornece a capacidade de estruturar, rotular e aprimorar os dados recebidos usando os serviços do [!DNL Platform]. Você pode assimilar dados de várias fontes, como aplicativos Adobe, armazenamento baseado em nuvem, bancos de dados e muitas outras.

O Experience Platform fornece suporte para assimilação de dados do sistema local, permitindo que você traga arquivos locais para o Experience Platform por meio do uso de conectores de origem.
