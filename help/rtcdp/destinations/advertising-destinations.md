# Destinos de publicidade {#advertising-destinations}

Conecte sua Plataforma de dados do cliente em tempo real a uma plataforma de publicidade de terceiros (por exemplo, DSP, rede de anúncios, SSP) e compartilhe públicos pseudônimos para essas plataformas.

Ao se conectar a um destino de publicidade, seus públicos-alvo são enviados como IDs para a plataforma de destino, onde são mapeados para uma ID conhecida pela plataforma de destino.

Atualmente, a CDP em tempo real suporta os destinos de publicidade listados abaixo. Para obter informações sobre como se conectar a esses destinos, consulte os links individuais:

* [Google Display &amp; Video 360](/help/rtcdp/destinations/google-dv360-destination.md)
* [Google Ads](/help/rtcdp/destinations/google-ads-destination.md)
* [Google Ad Manager](/help/rtcdp/destinations/google-ad-manager-destination.md)
