# (Beta) Conector IBM DB2

A Adobe Experience Platform fornece conectividade nativa para provedores de banco de dados como Microsoft, MySQL e Azure. Você pode trazer seus dados desses sistemas para a Plataforma.

Diferentes tipos de bancos de dados de terceiros são suportados, incluindo depósitos relacionais, NoSQL ou de dados. O suporte para provedores de banco de dados inclui o IBM DB2.

A documentação abaixo fornece informações sobre como conectar o IBM DB2 à Plataforma usando APIs ou a interface do usuário:
