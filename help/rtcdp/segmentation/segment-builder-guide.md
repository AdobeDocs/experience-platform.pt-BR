# Construtor de segmentos no Real-time Customer Data Platform

Criado com base na Adobe Experience Platform, [!DNL Adobe Real-Time Customer Data Platform] A pode usar todos os recursos do Construtor de segmentos que fazem parte da [!DNL Experience Platform]. Na interface do usuário do Construtor de segmentos, é possível criar novos segmentos, acessar segmentos existentes e modificar segmentos existentes.

Para obter um guia mais detalhado sobre o Construtor de segmentos, leia o [Guia do usuário do Construtor de segmentos](../../segmentation/ui/segment-builder.md).
