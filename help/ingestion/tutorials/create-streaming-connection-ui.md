# Criar uma conexão de transmissão usando a interface do usuário

Para iniciar o streaming de dados no [!DNL Experience Platform], primeiro você deve criar uma conexão HTTP de transmissão. Ao criar uma conexão de transmissão, é necessário fornecer detalhes importantes, como a fonte de dados de transmissão, e se você pretende enviar dados de uma fonte confiável (autenticada) ou não confiável (não autenticada).

Após registrar uma conexão de transmissão, você terá um URL exclusivo para o qual poderá ser usado para transmitir dados [!DNL Platform].

Para saber como criar uma conexão de transmissão, leia o [Guia da interface do usuário de origem da API HTTP](../../sources/tutorials/ui/create/streaming/http.md).
