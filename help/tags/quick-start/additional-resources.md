# Recursos adicionais

>[!NOTE]
>
>O Adobe Experience Platform Launch está sendo reformulado como um conjunto de tecnologias de coleção de dados na Experience Platform. Como resultado, várias alterações de terminologia foram implementadas na documentação do produto. Consulte o seguinte [documento](../term-updates.md) para obter uma referência consolidada das alterações de terminologia.

A comunidade de tags do Adobe Experience Platform está cheia de profissionais talentosos que compartilham dicas, tutoriais e ferramentas úteis na Web. Esta é uma coleção de alguns dos melhores recursos fornecidos pela comunidade até o momento.

## Uso geral

O repositório GitHub [Awesome Adobe Launch](https://github.com/MisterPhilip/awesome-adobe-launch) fornece uma lista de materiais com curadoria da comunidade relacionados às tags.
