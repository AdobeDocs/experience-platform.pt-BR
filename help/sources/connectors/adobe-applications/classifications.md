# Conector de origem do Adobe Analytics para dados de classificações

O Adobe Experience Platform permite que os dados sejam assimilados de fontes externas e, ao mesmo tempo, fornece a capacidade de estruturar, rotular e aprimorar os dados recebidos usando os serviços do [!DNL Platform]. Você pode assimilar dados de várias fontes, como aplicativos Adobe, armazenamento baseado em nuvem, bancos de dados e muitas outras.

O [!DNL Experience Platform] fornece suporte para assimilação de dados de Classificações do Analytics na Adobe Experience Platform.

As classificações permitem que os dados de variáveis do Analytics sejam categorizados e exibidos de diferentes maneiras quando os relatórios são gerados. O Conector de dados de classificações do Analytics requer que seus dados tenham sido migrados para a nova infraestrutura do [!DNL Classifications] do Adobe Analytics antes de serem usados. Para confirmar o status de migração dos dados, entre em contato com a equipe de conta do Adobe.
