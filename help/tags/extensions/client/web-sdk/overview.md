# Visão geral da extensão de tag do Adobe Experience Platform Web SDK

A extensão de tag do Adobe Experience Platform Web SDK envia dados para a Adobe Experience Cloud a partir das propriedades da Web, por meio do Adobe Experience Platform Edge Network. A extensão Experience Platform Web SDK permite a transmissão de dados na plataforma, sincronização de identidades, aceitação e coleta automática de dados de contexto.

Consulte [Introdução](getting-started.md) para saber como configurar e implantar esta extensão de tag para começar a coletar dados.
