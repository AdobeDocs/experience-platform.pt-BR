# Fontes no Real-Time Customer Data Platform

O [!DNL Adobe Real-Time Customer Data Platform] permite assimilar dados de fontes externas e fornece a capacidade de estruturar, rotular e aprimorar os dados recebidos usando os serviços do [!DNL Experience Platform]. Você pode assimilar dados de várias fontes, como aplicativos da Adobe, armazenamento baseado em nuvem, software de terceiros, seu sistema de CRM e muito mais.

Para obter uma visão geral mais detalhada das fontes, incluindo uma lista das fontes disponíveis e links para tutoriais, leia a [documentação das fontes](../../sources/home.md).
