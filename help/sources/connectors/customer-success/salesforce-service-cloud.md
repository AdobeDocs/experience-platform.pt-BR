# (Beta) Conector da Salesforce Service Cloud

>[!NOTE]
>O conector da Salesforce Service Cloud está em beta. Consulte a visão geral [das](../../home.md#terms-and-conditions) Fontes para obter mais informações sobre o uso de conectores com rótulo beta.

O Adobe Experience Platform permite que os dados sejam ingeridos de fontes externas e, ao mesmo tempo, fornece a você a capacidade de estruturar, rotular e aprimorar os dados recebidos usando os serviços Platform. Você pode assimilar dados de várias fontes, como aplicativos da Adobe, armazenamentos baseados em nuvem, bancos de dados e muitas outras.

O Experience Platform oferece suporte para a assimilação de dados de um sistema bem-sucedido de terceiros. O suporte para provedores bem-sucedidos do cliente inclui a Salesforce Service Cloud.
