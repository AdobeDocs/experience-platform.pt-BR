# (Beta) Conector Apache HDFS

A plataforma Adobe Experience fornece conectividade nativa para provedores de nuvem como AWS, plataforma Google Cloud e Azure, permitindo que você traga seus dados desses sistemas. Os dados ingeridos podem ser formatados como JSON, parquet ou delimitados. O suporte para provedores de armazenamentos em nuvem inclui o Apache HDFS.

A documentação abaixo fornece informações sobre como conectar o Apache HDFS à plataforma usando APIs ou a interface do usuário:

- [Coletar dados de armazenamento na nuvem usando a API de Serviço de Fluxo](../../tutorials/api/collect/cloud-storage.md)
