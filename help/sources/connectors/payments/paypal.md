# Conector (Beta) [!DNL PayPal]

>[!NOTE]
>O [!DNL PayPal] conector está em beta. Consulte a visão geral [das](../../home.md#terms-and-conditions) Fontes para obter mais informações sobre o uso de conectores com rótulo beta.

O Adobe Experience Platform permite que os dados sejam assimilados de fontes externas, ao mesmo tempo em que lhe fornece a capacidade de estruturar, rotular e aprimorar os dados recebidos usando [!DNL Platform] serviços. Você pode assimilar dados de várias fontes, como aplicativos da Adobe, armazenamentos baseados em nuvem, bancos de dados e muitas outras.

[!DNL Experience Platform] fornece suporte para assimilar dados de um aplicativo de pagamentos de terceiros. O apoio aos prestadores de pagamentos inclui [!DNL PayPal].
