# Visão geral do Brightscript

O **Adobe Experience Platform Roku SDK** permite enviar dados para uma [sequência de dados](/help/datastreams/overview.md) de um dispositivo Roku usando Brightscript.

A SDK e a documentação dessa biblioteca estão hospedadas no GitHub:

**[Adobe Experience Platform Roku SDK](https://github.com/adobe/aepsdk-roku)**
