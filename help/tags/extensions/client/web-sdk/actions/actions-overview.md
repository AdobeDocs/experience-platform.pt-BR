# Tipos de ação

Esta página lista todas as ações compatíveis em uma regra de tag. Para criar ou editar as ações em uma regra de tag:

1. Faça logon em [experience.adobe.com](https://experience.adobe.com) usando suas credenciais da Adobe ID.
1. Navegue até **[!UICONTROL Data Collection]** > **[!UICONTROL Tags]**.
1. Selecione a propriedade de tag desejada.
1. Navegue até **[!UICONTROL Rules]** e selecione a regra desejada.
1. Em [!UICONTROL Actions], selecione uma ação existente ou crie uma ação.
1. Defina o campo suspenso [!UICONTROL Extension] como **[!UICONTROL Adobe Experience Platform Web SDK]** e selecione o [!UICONTROL Action Type] desejado.

A extensão de tag do Web SDK oferece as seguintes ações:

* [**[!UICONTROL Apply propositions]**](apply-propositions.md): Renderiza apresentações em aplicativos de página única sem incrementar métricas.
* [**[!UICONTROL Apply response]**](apply-response.md): Executar uma ação baseada em uma resposta do Edge Network.
* [**[!UICONTROL Evaluate rulesets]**](evaluate-rulesets.md): acionar manualmente uma avaliação de conjunto de regras.
* [**[!UICONTROL Get Media Analytics tracker]**](get-media-analytics-tracker.md): exporte a API de mídia herdada para um objeto de janela.
* [**[!UICONTROL Redirect with identity]**](redirect-with-identity.md): permite o compartilhamento de um identificador de visitante entre domínios que você possui.
* [**[!UICONTROL Send event]**](send-event.md): Envia dados do evento para a Edge Network.
* [**[!UICONTROL Send media event]**](send-media-event.md): Envia dados de mídia para a Edge Network.
* [**[!UICONTROL Set consent]**](set-consent.md): define o consentimento desejado para o visitante.
* [**[!UICONTROL Update variable]**](update-variable.md): modifica um elemento de dados de [variável](../data-element-types.md#variable).
