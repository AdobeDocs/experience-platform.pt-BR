# `personalization`

A variável `personalization` O objeto contém várias propriedades que permitem adaptar a experiência do usuário dependendo dele. Ele é especialmente valioso em implementações do Adobe Target ou do Adobe Journey Optimizer, pois é a força motriz que permite personalizar o conteúdo exibido por usuário.

Consulte a documentação sobre como [renderizar conteúdo personalizado](../../personalization/rendering-personalization-content.md) para obter mais informações.
