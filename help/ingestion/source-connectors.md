# Conectores de origem

O Adobe Experience Platform permite que os dados sejam assimilados de fontes externas, fornecendo a capacidade de estruturar, rotular e aprimorar os dados recebidos usando serviços [!DNL Platform]. Você pode assimilar dados de várias fontes, como aplicativos Adobe, armazenamento baseado em nuvem, software de terceiros e seu CRM.

[!DNL Experience Platform] O fornece uma RESTful API e uma interface interativa que permite configurar conexões de origem para vários provedores de dados com facilidade. Essas conexões de origem permitem autenticar seus sistemas de armazenamento e serviços de CRM, definir horários para execuções de assimilação e gerenciar a taxa de transferência de assimilação de dados.

Mais informações sobre esses conectores de origem podem ser encontradas na [documentação dos conectores de origem](../sources/home.md).
