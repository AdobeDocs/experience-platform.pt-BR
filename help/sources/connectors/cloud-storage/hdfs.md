# (Beta) Conector Apache HDFS

>[!NOTE]
>O conector HDFS Apache está em beta. Consulte a visão geral [das](../../home.md#terms-and-conditions) Fontes para obter mais informações sobre o uso de conectores com rótulo beta.

O Adobe Experience Platform fornece conectividade nativa para provedores de nuvem como AWS, Google Cloud Platform e Azure, permitindo que você traga seus dados desses sistemas. Os dados ingeridos podem ser formatados como JSON, parquet ou delimitados. O suporte para provedores de armazenamentos em nuvem inclui o Apache HDFS.

A documentação abaixo fornece informações sobre como conectar o Apache HDFS à Platform usando APIs ou a interface do usuário:
