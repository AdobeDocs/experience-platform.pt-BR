# Mapeamentos do campo de dados meteorológicos

A Adobe fez uma parceria com o [[!DNL [The Weather Company]]](https://www.ibm.com/weather) para trazer o contexto adicional do clima dos Estados Unidos para os dados coletados por sequências de dados. Você pode usar esses dados para análise, direcionamento e criação de segmentos na Experience Platform.

Esta página lista todos os campos disponíveis que você pode usar para enriquecer os dados do público-alvo.

Os campos são divididos em três grupos diferentes que se alinham aos grupos de campos.

* [**[!UICONTROL Current Weather]**](#current-weather): as condições meteorológicas atuais do usuário, com base em sua localização. Inclui a temperatura atual, a precipitação, a cobertura de nuvens e muito mais.
* [**[!UICONTROL Forecasted Weather]**](#forecast): A previsão inclui a previsão de 1,2,3,5,7 e 10 dias para o local do usuário.
* [**[!UICONTROL Triggers]**](#triggers): Os acionadores são combinações específicas que mapeiam para condições meteorológicas semânticas diferentes. Existem três tipos diferentes de acionadores meteorológicos:

   * **[!UICONTROL Relative Triggers]**: Condições semanticamente significativas, como tempo frio ou chuvoso. Essas definições podem diferir em vários climas.
   * **[!UICONTROL Product Triggers]**: Condições que levariam à compra de diferentes tipos de produtos. Por exemplo: as previsões de tempo frio podem significar que as compras de casacos de chuva serão mais prováveis.
   * **[!UICONTROL Severe Weather Triggers]**: avisos de tempo severo, como avisos de tempestade de inverno ou de furacão.

## Acionadores {#triggers}

Os acionadores definem a semântica das condições meteorológicas com base em várias entradas.

| Campo | Descrição | Caminho XDM |
| --- | ---- | --- |
| [!UICONTROL Product Triggers] | Indicam quando as condições são adequadas para impulsionar as vendas de determinadas categorias de produtos. Eles são mapeados para identidades de inteiros. Uma lista completa pode ser obtida da IBM. | `weather.productTriggers` |
| [!UICONTROL Relative Triggers] | Condições relativas baseadas na percepção humana. Aspectos como “quente” ou “frio”. Eles são mapeados para identidades de inteiros. Uma lista completa pode ser obtida da IBM. | `weather.relativeTriggers` |
| [!UICONTROL Severe Weather Triggers] | Indica várias condições climáticas severas, como furacões ou excesso de chuva. | `weather.severeTriggers` |
