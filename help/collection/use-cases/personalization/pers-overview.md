# Visão geral dos casos de uso do Personalization

O Adobe Experience Platform Web SDK permite que as empresas forneçam uma grande variedade de casos de uso de personalização a suas propriedades da Web. Você pode conectar facilmente os processos de coleta, decisão e ativação de dados em soluções da Adobe, como Target, Journey Optimizer e Audience Manager.

O SDK é compatível com arquiteturas flexíveis, incluindo modelos híbridos, do lado do cliente e do lado do servidor, permitindo buscar e renderizar ofertas personalizadas que atendam à maioria dos casos de uso da organização.

## Amostras do Web SDK

Além das páginas de documento nessa pasta, o Adobe mantém um repositório de aplicativos de amostra que podem ser referenciados. Consulte [amostras do Web SDK](https://github.com/adobe/alloy-samples/) no GitHub para obter mais casos de uso de personalização, incluindo:

* Personalização do lado do cliente
* Personalização do lado do servidor
* Personalização híbrida
* Personalization usando aplicativos de página única
