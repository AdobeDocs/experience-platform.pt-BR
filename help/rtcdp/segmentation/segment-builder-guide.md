# Construtor de segmentos no Real-time Customer Data Platform

Criado com base no Adobe Experience Platform, [!DNL Real-time Customer Data Platform] O pode utilizar todos os recursos do Construtor de segmentos que fazem parte do [!DNL Experience Platform]. Na interface do usuário do Construtor de segmentos, é possível criar novos segmentos, acessar segmentos existentes e modificar segmentos existentes.

Para obter um guia mais detalhado sobre o Construtor de segmentos, leia o [Guia do usuário do Construtor de segmentos](../../segmentation/ui/segment-builder.md).
