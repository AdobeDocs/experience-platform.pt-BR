# Conector HDFS (Beta)

A plataforma Adobe Experience fornece conectividade nativa para provedores de nuvem como AWS, plataforma Google Cloud e Azure, permitindo que você traga seus dados desses sistemas. Os dados ingeridos podem ser formatados como JSON, parquet ou delimitados. O suporte para provedores de armazenamentos em nuvem inclui o HDFS.

A documentação abaixo fornece informações sobre como conectar o HDFS à plataforma usando APIs:
