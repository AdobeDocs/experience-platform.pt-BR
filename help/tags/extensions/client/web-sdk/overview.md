# Visão geral da extensão do SDK da Web da Adobe Experience Platform

>[!NOTE]
>
>O Adobe Experience Platform Launch foi reformulado como um conjunto de tecnologias de coleta de dados na Adobe Experience Platform. Como resultado, várias alterações de terminologia foram implementadas na documentação do produto. Consulte o seguinte [documento](../../../term-updates.md) para obter uma referência consolidada das alterações de terminologia.

A extensão SDK da Web da Adobe Experience Platform envia dados para a Adobe Experience Cloud de propriedades da Web, por meio da rede de borda da Adobe Experience Platform. A extensão Experience Platform Web SDK permite a transmissão de dados na plataforma, sincronização de identidades, aceitação e coleta automática de dados de contexto.

Para obter detalhes sobre como configurar a extensão do Experience Platform Web SDK, consulte a [Configurar a extensão de tag do Web SDK](web-sdk-extension-configuration.md).
