# (Beta) [!DNL Apache Hive] no [!DNL Azure HDInsights] conector

>[!NOTE]
>O Apache Hive no conector HDInsights do Azure está em beta. Consulte a visão geral [das](../../home.md#terms-and-conditions) Fontes para obter mais informações sobre o uso de conectores com rótulo beta.

O Adobe Experience Platform permite que os dados sejam assimilados de fontes externas, ao mesmo tempo em que lhe fornece a capacidade de estruturar, rotular e aprimorar os dados recebidos usando [!DNL Platform] serviços. Você pode assimilar dados de várias fontes, como aplicativos da Adobe, armazenamentos baseados em nuvem, bancos de dados e muitas outras.

[!DNL Experience Platform] fornece suporte para assimilar dados de um banco de dados de terceiros. [!DNL Platform] pode se conectar a diferentes tipos de bancos de dados, como relacional, NoSQL ou datas warehouses. O suporte para provedores de banco de dados inclui [!DNL Apache Hive] o [!DNL Azure HDInsights].
