# Visão geral do Data Distiller

O Data Distiller é uma oferta de pacote que inclui um subconjunto das funcionalidades do Adobe Experience Platform. Com o Data Distiller, você pode executar a preparação de dados após a assimilação (como limpeza, modelagem e manipulação) para o Perfil do cliente em tempo real ou casos de uso analíticos, executando consultas em lote no Serviço de consulta. Você também pode [criar modelos de dados de relatórios personalizados](../../query-service/data-distiller/customizable-insights/reporting-insights-data-model.md) ou [usar o SQL para gerar gráficos para seus painéis personalizados](./customizable-insights/overview.md).

>[!NOTE]
>
>Seu uso do Data Distiller depende de seus direitos para aplicativos baseados em plataforma.
