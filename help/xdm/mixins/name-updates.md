# Misturar atualizações de nomes

Os nomes de várias combinações do Experience Data Model (XDM) foram alterados. Uma vez que a estrutura de campo destas misturas permanece a mesma, as alterações de nome por si só não devem afetar os esquemas existentes que empregam estas misturas. No entanto, para procurar e usar essas combinações na interface do usuário do Adobe Experience Platform, os novos nomes devem ser usados.
