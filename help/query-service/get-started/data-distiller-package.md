# Pacote do Data Distiller

O Data Distiller é uma oferta de pacote que inclui um subconjunto das funcionalidades do Adobe Experience Platform. Este pacote destina-se à preparação de dados após a assimilação para casos de uso de perfil do cliente em tempo real ou dados analíticos por consultas em lote executadas no Serviço de consulta. Consulte a [Visão geral do Data Distiller](../data-distiller/overview.md) para obter mais detalhes sobre o pacote Data Distiller.
